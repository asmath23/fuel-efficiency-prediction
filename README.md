# 🚗 Fuel Efficiency Prediction using Polynomial Regression

## 📌 Project Overview
This project analyzes the relationship between vehicle speed and fuel efficiency using data analysis and machine learning.

## 📊 Key Steps
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Visualization (Scatter Plot, Histogram, Boxplot)
- Polynomial Regression Model
- Model Evaluation (R², MSE, RMSE)

## 📈 Results
- R² Score: ~0.98 (Train & Test)
- Model shows strong generalization
- Captures non-linear relationship effectively

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 🎯 Conclusion
Fuel efficiency shows a non-linear relationship with speed. Polynomial regression effectively models this pattern and provides accurate predictions.
